# Powerbi_dashboardDashboard 
A walkthrough of the dashboard's components, including KPIs like total salary, basic salary, allowances, deductions, and net pay, along with trends, growth comparisons, and departmental breakdowns. It also highlights sections for preferred payment methods by employment type, employee count by department, and top five employees by basic salary and allowances.
Data Loading and Setup : Demonstrates how to load salary data into Power BI, rename tables, and set up the initial page layouts for "Overview" and "Employees."
Calendar Table Creation and Linking : Explains how to create a calendar table to manage date-related calculations and link it to the main salary data table for proper trend analysis.
Measure Creation (Basic Salary) : Details the process of creating measures for basic salary, previous month basic salary, and advanced calculations for month-over-month growth, including handling scenarios where no month is selected.
Visual Formatting and Interactivity : Shows how to format visuals such as cards and line charts, add forecasting, adjust colors, and manage interactions between different visuals to prevent unintended filtering.
Populating and Formatting Remaining Visuals : Guides through adding allowances, deductions, and net pay visuals, applying conditional formatting for colors, and setting up visual properties like rounded edges and trend lines.
Adding Key Information and Filters : Demonstrates how to include total salary and salary period information, and how to add filters for month and employee selections.
Payment History and Preferred Mode Visuals : Explains how to create visuals for payment history by department, preferred mode of payment, and how payment methods are partitioned by employment type.
Employee Metrics Visuals : Covers creating visuals for the total number of employees and employees by department, broken down by employment type.
Top Five Employees Analysis: Shows how to identify and display the top five employees based on selected metrics (basic salary, allowances, deductions, net pay) using a field parameter for dynamic selection.
